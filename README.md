## 📖 Overview
Document Q&A Chatbot is an end-to-end **Retrieval-Augmented Generation (RAG)** system that enables users to upload documents (PDF, DOCX, TXT), ask natural language questions, and receive accurate, context-aware responses generated by a **QLoRA fine-tuned Meta Llama 3.2 (3B)** model.

Unlike traditional document search systems, this project combines:
- Dense semantic retrieval
- Context-aware generation
- Parameter-efficient fine-tuning
- FastAPI backend
- Streamlit interface
- Chroma Vector Database
- SentenceTransformer embeddings

to create an efficient and scalable document intelligence platform.

## 📚 Dataset Preparation
Generate the instruction tuning dataset.
```bash
cd data
python prepare_dataset.py \
    --train_size 5000 \
    --eval_size 300
cd ..
```
This script:
- Downloads CUAD dataset
- Cleans records
- Removes invalid samples
- Converts data into instruction format
- Creates train/eval JSONL files

---

## 🧠 Fine-Tuning
Train using QLoRA.
```bash
cd finetune
python train_lora.py \
    --base_model meta-llama/Llama-3.2-3B-Instruct \
    --train_file ../data/train.jsonl \
    --output_dir ./lora-doc-qa \
    --epochs 3
```

**Training configuration**
| Parameter | Value |
|---|---|
| Quantization | 4-bit NF4 |
| LoRA Rank | 16 |
| Alpha | 32 |
| Dropout | 0.05 |
| Optimizer | AdamW |
| Precision | bfloat16 |
| Framework | PEFT + TRL |

---

## 📊 Model Evaluation
Evaluate the fine-tuned model.
```bash
python evaluate.py \
    --base_model meta-llama/Llama-3.2-3B-Instruct \
    --adapter_dir ./lora-doc-qa \
    --eval_file ../data/eval.jsonl
```
Metrics:
- Exact Match (EM)
- F1 Score
- ROUGE-L

Results are stored in `eval_results.json`.

## 🔍 RAG Pipeline

### 1. Document Upload
Supports: PDF, DOCX, TXT

### 2. Chunking
- Token-based chunking
- 512 token size
- 64 token overlap

### 3. Embedding Generation
Uses SentenceTransformer to convert chunks into dense vectors.

### 4. Vector Storage
Vectors stored inside ChromaDB.

### 5. Semantic Retrieval
Top-K similarity search.

### 6. Context Generation
Retrieved chunks combined into prompt.

### 7. Answer Generation
Fine-tuned Llama generates final response.
